MatzPhone

Status: pre-alpha — scaffolding & bring-up. No flashable image yet.

An open-source, privacy-first mobile operating system built on AOSP — developed in public, with the long-term goal of hardware that ships with it out of the box.

Source on GitHub Roadmap Security policy

No bundled tracking

No Google services baked in. Telemetry off. MicroG and sandboxed Play Services are opt-in, never preinstalled.

Long support windows

Target: 5+ years of security updates, with monthly bulletin fixes shipped within 7 days of publication.

Verifiable builds

Reproducible-build CI is on the roadmap so anyone can confirm the shipped image matches the public source tree.

Owner-controlled keys

Verified boot with a user-settable root of trust — re-lock your bootloader with your own keys.

Where the project actually is

  1. Phase 0 — scaffolding (now): build manifest layered on pinned AOSP, distribution vendor tree, CI including nightly manifest-rot checks.
  2. Phase 1: first booting build on a Pixel-class reference device, signed test builds.
  3. Phase 2: daily-drivable alpha, OTA pipeline, reproducible builds.
  4. Phase 3: hardware — starting with preflashed flagship devices, factory preloads as the project earns them.
Apache-2.0. Built in public — the work backlog is part of the repo. Contributions welcome; see CONTRIBUTING.
